Yayınlama Notları

  1. Bu paket statik ve bağımsızdır. Harici kütüphane/bağımlılık yoktur.
  2. UTF-8 ayarlı; Türkçe karakter problemi yaşamaz.
  3. GitHub Pages için: yeni repo → index.html köke yükle → Settings → Pages → Deploy from a branch → main / root.
  4. Netlify için: New site from Git veya ZIP yükleme → build komutu gerekmez.
  5. Vercel için: Proje ekle → Output directory: /.

Sık Görülen Hata ve Çözümler

  • “This file is empty”: Yüklediğiniz ZIP boştur veya klasörün içi boş yüklenmiştir. Bu pakette index.html vardır; ZIP’i doğrudan yükleyin.
  • 404 Not Found: Pages ayarlarında Branch ve Root doğru seçildi mi?
  • Türkçe karakterler bozuk: <meta charset="utf-8"> var; tarayıcı önbelleğini temizleyin.
